Enable capture on release (TRUE to enable or FALSE to disable). The default is FALSE. [Get and set.] TRUE : The result callback will be called when the user releases fingers from the sensor. FALSE : The result callback will be called when the quality of the finger is good

ENUM_IBSU_PROPERTY_ENABLE_SPOOF 

Enable spoof function The default is FALSE. [Get and set.]

ENUM_IBSU_PROPERTY_SPOOF_LEVEL 

Change spoof Level/Sensitivity The valid range is between 1 and 5. The default is 3. [Get and set.] The default sensitivity of 3 provides the best results through a range of test environments. The default value of 3 should be used as the baseline sensitivity when tuning application performance.

An INCREASE in spoof level sensitivity (> 3) will increase the number of false rejections for live fingers while decreasing the liklihood of false acceptance of spoof prints.

A DECREASE in spoof level sensitivity (< 3) will reduce false rejections but increases the liklihood of false acceptance of spoof prints.

Spoof sensitivity is the scale for algorithm thresholds that determines the liklihood that a fingerprint is fake, or a spoof. The feature is designed so that the algorithm returns a warning callback when a spoof is detected
