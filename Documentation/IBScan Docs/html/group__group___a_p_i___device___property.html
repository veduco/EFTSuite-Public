int WINAPI IBSU_GetProperty ( const int  handle,
const IBSU_PropertyId  propertyId,
LPSTR  propertyValue 
)

Retrieves a particular device's property value (by handle).

Precondition
IBSU_OpenDevice() or IBSU_OpenDeviceEx() or IBSU_AsyncOpenDevice() called successfully
Parameters
[in]handleDevice handle obtained by OpenDevice functions.
[in]propertyIdProperty identifier to get value for.
[out]propertyValueString returning property value. Memory must be provided by a caller. This buffer should be able to hold IBSU_MAX_STR_LEN characters.
Returns
IBSU_STATUS_OK, if successful.
Error code < 0, otherwise.
See also
See error codes in 'IBScanUltimateApi_err.h'.
