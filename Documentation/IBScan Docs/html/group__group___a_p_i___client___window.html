If ENUM_IBSU_OVERLAY_SHAPE_ARROW, reserved_1 should be the width (in pixels) of the full base of the arrowhead, and reserved_1 should be the angle (in radians) at the arrow tip between the two sides of the arrowhead. x1 X coordinate of start point of overlay shape. y1 Y coordinate of start point of overlay shape. x2 X coordinate of end point of overlay shape. y2 Y coordinate of end point of overlay shape. lineWidth Line width. lineColor Line color. reserved_1 X reserved. reserved_2 Y reserved.
Returns
IBSU_STATUS_OK, if successful.
